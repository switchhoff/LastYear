# Year Ago Today

This project is a nostalgic reminder of fun events that happened exactly one year ago.

## GitHub Repository
Connected to: [https://github.com/switchhoff/LastYear.git](https://github.com/switchhoff/LastYear.git)

## Getting Started

To finalize the connection to your GitHub repository and push your code, run the following commands in your terminal:

```bash
# Remove the existing remote if it exists
git remote remove origin

# Add the remote repository using HTTPS
git remote add origin https://github.com/switchhoff/LastYear.git

# Initial push
git add .
git commit -m "Connect to GitHub via HTTPS and update project metadata"
git push -u origin main
```

## Features
- Shared memories with chat and reactions.
- Automated date calculations.
- Nostalgic daily updates.
