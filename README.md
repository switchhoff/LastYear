# Year Ago Today

This project is a nostalgic reminder of fun events that happened exactly one year ago.

## GitHub Repository
Connected to: [git@github.com:switchhoff/LastYear.git](git@github.com:switchhoff/LastYear.git)

## Getting Started

To finalize the connection to your GitHub repository and push your code, run the following commands in your terminal:

```bash
# Add the remote repository
git remote add origin git@github.com:switchhoff/LastYear.git

# Initial push
git add .
git commit -m "Connect to GitHub and update project metadata"
git push -u origin main
```

## Features
- Shared memories with chat and reactions.
- Automated date calculations.
- Nostalgic daily updates.
