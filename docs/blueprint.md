# **App Name**: Year Ago Today

## Core Features:

- Date Calculation: Automatically calculate the date exactly one year prior to the current date.
- Text Display: Display the calculated date and a user-defined sentence prominently on the screen.
- AI Stick Figure Art: Use generative AI to create a stick figure art piece related to the user-defined sentence to appear above the date and sentence. Use prompt engineering tools to generate good, appropriate stick figure illustrations.
- Daily Update: Automatically update the date, sentence, and stick figure art every day.

## Style Guidelines:

- Background color: Very dark grayish-blue (#1A1A1C) to provide contrast and a subtle backdrop for the text and artwork.
- Primary color: Pale blue (#ADD8E6) for the date and sentence, ensuring readability against the dark background.
- Accent color: Light cyan (#E0FFFF) to highlight elements and draw attention.
- Font: 'Inter' sans-serif for both date/time and body, offering a clean and modern look.
- Stick figure art generated by AI should have a simple, clean style.
- Simple layout with the date and sentence centered, and AI art placed above. Minimalist design.
- Subtle fade-in animation when the date, sentence, and art update each day.
